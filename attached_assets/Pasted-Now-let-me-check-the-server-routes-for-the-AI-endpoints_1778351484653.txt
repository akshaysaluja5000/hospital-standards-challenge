### 🔴 Bug 1 (Critical): Wrong Anthropic Model String — All AI Features Broken

**File:** `server/routes.ts` (all 8 AI endpoints)

Every AI call uses `"claude-haiku-4-5"` but the correct API model string is **`"claude-haiku-4-5-20251001"`**. The abbreviated form is not a recognized model ID and will cause `invalid_request_error` responses from the Anthropic API, breaking:
- AI Tutor (wrong answers explanation)
- AI Debrief
- AI Handbook Search
- Topic Quiz Modal
- Admin AI Insights
- AI Leadership Coach

**Fix:** Replace all 8 instances of `"claude-haiku-4-5"` → `"claude-haiku-4-5-20251001"`
